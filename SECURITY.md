# Security Policy

SliceFx is experimental and has not had a formal security review.

## Supported versions

Security fixes are handled on the default branch until the project begins publishing stable releases. Once stable packages exist, this file will be updated with supported version ranges.

## Reporting a vulnerability

Please do not report security vulnerabilities in public issues.

Use GitHub's private vulnerability reporting for this repository when available. If private reporting has not been enabled yet, open a public issue asking for a private contact channel without including exploit details.

We aim to acknowledge reports within 7 days and will coordinate disclosure timing with the reporter.
